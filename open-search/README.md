# How to Start
**Run as administrator**
```
docker-compose up
```
**Open on your preferred browser**
```
http://localhost:5601/
```
**Default Username/Password**
```
admin/admin
```
**Populate an Elasticsearch index from bash and json files**
```
sudo bash index-data.sh
```
